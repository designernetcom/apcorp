We are the best supplier of Control Panels.

There are Jalganga control panels available for submerged pump sets that are single or three phase. Capacitors for beginning and running are found in panels. While running capacitors guarantee smooth motor operations, starting capacitors supply the torque needed to start the motor. Depending on the motor rating, a range of start and run capacitors is offered. A voltage tester is also included for monitoring the input voltage.
